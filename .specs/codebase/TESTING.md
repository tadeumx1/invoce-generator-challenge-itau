# Testing Infrastructure

## Test Frameworks

- **Unit / Integration:** JUnit Jupiter 5 via `spring-boot-starter-test` 3.5.14.
- **Mocking:** no Mockito usage in the current tests; Mockito is excluded from the test starter to avoid unnecessary JVM self-attach behavior.
- **Spring Test:** `@SpringBootTest` + MockMvc for application and HTTP integration tests.
- **Coverage:** JaCoCo 0.8.11.
- **Format/style:** Spotless + google-java-format, Checkstyle import policy.

## Test Organization

**Location:** `src/test/java/br/com/itau/invoicegenerator/`.

**Main groups:**

- `GenerateInvoiceInteractorTest` verifies use-case orchestration with a recording fake.
- `service/TaxRateSelection*Test` covers tax brackets.
- `service/FreightMultiplierTest` covers freight multipliers.
- `service/characterization/*Test` now contains regression tests for fixed C-1/C-2/C-3 behavior plus the remaining slow C-6 characterization.
- `web/InvoiceControllerIntegrationTest` verifies HTTP JSON contract with sample payloads.
- `InvoiceGeneratorApplicationTests` verifies Spring context wiring.

## Test Execution

```bash
./mvnw test                                      # fast suite, excludes @Tag("slow")
./mvnw test -Pslow                              # slow characterization suite
./mvnw test -Dtest=GenerateInvoiceInteractorTest
./mvnw verify                                   # tests + Spotless + Checkstyle + JaCoCo
```

**JDK requirement:** Java 21.

## Coverage Targets

**Current:** JaCoCo report generated by `./mvnw verify`.
**Goal:** add enforcement later, with at least 80% on domain/application and 60% on adapters.

## Gate Check Commands

| Gate Level | When to Use | Command |
| --- | --- | --- |
| Quick | Local implementation loop | `./mvnw test` |
| Slow characterization | Changes near delivery/integration latency | `./mvnw test -Pslow` |
| Build | Phase completion / pre-commit | `./mvnw verify` |

`./mvnw verify` runs tests, Spotless check, Checkstyle, jar packaging, and JaCoCo report generation.
